# Loopwish · Apple

![Loopwish banner](https://raw.githubusercontent.com/loopwish/shared/main/assets/logos/loopwish-banner.png)

Loopwish er en ønskeliste-app som lar deg dele ønsker med familie og venner. Ønsk. Del. Få. Sammen.

Dette repoet inneholder Apple-klienten for Loopwish.

## Plattformstøtte

- iOS 16+
- iPadOS 16+
- macOS 13+

## Teknologi-stack

- Swift
- SwiftUI
- Swift Package Manager (SPM)
- SwiftLint

## Kom i gang

### Forutsetninger

- Xcode 15+ (anbefalt)
- Swift 5.9+
- SwiftLint (CI installerer automatisk)

### Installering

```bash
git clone https://github.com/loopwish/apple.git
cd apple
```

### Kjør tester

```bash
swift test
```
